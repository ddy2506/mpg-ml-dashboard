# Car MPG Machine Learning Dashboard

This project demonstrates a complete data analysis and machine learning pipeline using Python.

## Features
- Loads and cleans real-world car fuel efficiency data
- Visualizes data with bar charts and histograms
- Trains a simple machine learning model to predict MPG from horsepower
- Saves charts as PNG files for reporting
- Provides an interactive Streamlit web app for real-time prediction

## Technologies Used
- Python
- pandas
- matplotlib
- numpy
- Streamlit
